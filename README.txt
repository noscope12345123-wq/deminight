Student Progress Tracker Pro Ultra is a Python coursework application created for module IY499. The program was designed to manage student data in a more advanced way than a simple mark book. It includes three interfaces: a console version, a Tkinter desktop version, and a Flask web version. The tracker allows the user to add students, modules, and weighted assessments, then analyse performance using averages, grade bands, attendance, pass rate, trend analysis, and a calculated risk score. It also generates recommendations to suggest support actions.

The project demonstrates practical programming topics from class, including functions and methods, file handling, error handling, searching, sorting, data structures, loops, regular expressions, and data visualisation. Student information is saved in JSON, reports can be exported to CSV, and activity is logged. A Tkinter Canvas chart is used to visualise student averages. The program also includes explicit linear search and bubble sort algorithms for higher marks. External libraries are managed through requirements.txt, and the code is organised into reusable modules. Test cases are included for valid, invalid, and edge-case scenarios.
